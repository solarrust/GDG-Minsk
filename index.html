web.dev. Logical Properties

Размеры

  • width → inline-size
  • height → block-size

+ min/max

                .selector {
                  width: 50%;
                  min-width: 150px;
                  max-width: 800px;
                  height: 50%;
                  min-height: 150px;
                  max-height: 800px;
                }
→ 
                .selector {
                  inline-size: 50%;
                  min-inline-size: 150px;
                  max-inline-size: 800px;
                  block-size: 50%;
                  min-block-size: 150px;
                  max-block-size: 800px;
                }
→ 
                .selector {
                  inline-size: clamp(150px, 50%, 800px);
                  block-size: clamp(150px, 50%, 800px);
                }
→

clamp()

                .selector {
                  inline-size: clamp(150px, 50%, 800px);
                  aspect-ratio: 1/1;
                }

aspect-ratio
